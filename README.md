# Inventario de  Frutería

Sistema interno de gestión de inventario desarrollado con Next.js 16, React 19, TypeScript estricto y SQLite.

La aplicación permite:

- Visualizar productos del inventario
- Calcular estadísticas financieras en tiempo real
- Actualizar stock persistente en SQLite
- Gestionar estados visuales de stock
- Aplicar descuentos automáticamente

---

# Tecnologías utilizadas

- Next.js 16 (App Router)
- React 19
- TypeScript 5 (`strict: true`)
- Tailwind CSS 4
- SQLite
- better-sqlite3
- sonner 2
